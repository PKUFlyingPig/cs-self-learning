# 国立台湾大学：林沛群机器人学

## 课程简介

- 所属大学：国立台湾大学
- 先修要求：线性代数和微积分
- 编程语言：无
- 课程难度：🌟🌟🌟
- 预计学时：35小时


这门课是 B 站上机器人学理论播放量最高的课程之一（可能没有之一，就是最高），主要内容是机器人学的基础理论，如坐标变换、关节表示、正逆运动学等，其分为运动学和动力学部分，对于研究机器人系统而非纯 AI 的人来说是一个很好的入门课程，老师讲解的非常清晰易懂
