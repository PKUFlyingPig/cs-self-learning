# The Information Theory, Patter Recognition, and Neural Networks

## 课程简介

- 所属大学：Cambridge
- 先修要求：Calculus, Linear Algebra, Probabilities and Statistics
- 编程语言：Anything would be OK, Python preferred
- 课程难度：🌟🌟🌟
- 预计学时：30-50 小时
